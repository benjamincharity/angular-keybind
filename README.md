## Installation

#### NPM
```bash
npm install angular-keybind --save
```

#### Bower
```bash
bower install angular-keybind --save
```

#### Manually

Add the script to your HTML:

```html
<script src="../path/to/your/(npm|bower)_components/angular-keybind/dist/angular-keybind.js"></script>
```

##Dependencies

- Angular.js (^1.4.0)
